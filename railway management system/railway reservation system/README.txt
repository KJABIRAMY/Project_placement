A GUI-based Railway Reservation System created with Java Swing.

Features:
- Visual seat layout display
- Booking and cancellation dialogs
- Real-time seat status updates
- Color-coded buttons
- Hover effects on buttons
- Scrollable booking information
- Status messages for user actions

To run:
1. Compile: javac RailwayReservationSystemGUI.java
2. Run: java RailwayReservationSystemGUI

The GUI includes:
- Top panel with title and available seats count
- Left panel with seat layout (organized by rows)
- Right panel with booking information
- Bottom panel with action buttons
